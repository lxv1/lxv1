22. In the  standard tableau, values of   variables are written on the top:


    0   1   2  3  4  -1

    2   0   0  1 -1  -1

   -1  -2   1  0 -1  -2

    0  -2  -1  0  3  -2

    0   0   0  0  0   0

 

This gives us  (A) an infeasible solution, (B) a infeasible but not optimal solution, (C) an  optimal solution, (D) the basic solution.

Solution. We compute he values for  the variables at the right margin:


    0   1   2  3  4  -1

    2   0   0  1 -1  -1  = 0

   -1  -2   1  0 -1  -2  =-2

    0  -2  -1  0  3  -2  =10

    0   0   0  0  0   0

 


A variable take a negative value, -10, so we get an infeasible  solution.

Thus, (A) is correct while .(C) and (C) are false.  (D) is false because all variables on top must be 0 for the basic solution.

Answer. A.

27.  The arbitration pair   for the 1 by 3  bimatrix game

  [(1, 2), (2, 3), (3, 2)] , is 

(A) (1, 3),    (B) (1, 2), (C)  (2, 3), (D)  (3,2).

Solution.  The mixtures of given 3 pure payoffs form a triangle with these payoffs as  as corners.,

The Pareto optimal solutions is the side of triangle formed by the mixtures of.   (2, 3), and (3, 2).

This side belongs to the line. x + y = 5.  The initial points us  (1, 3) (see #11),

So we maximize  (x - 1)(y - 3) on the line  (x -1)+ (y -3) = 1 and get x - 1 = 1/2 = y - 3, hence

x = 1.5 and y = 3.5.  This point is outside the side. The closest point on the side is. (2, 3).

So the   arbitration pair (x*, y*) is (2, 3).

(C).
