Let  X be a randon variable with  finite expectation for |X|.
Let  Xi be independent random variables with the same distribution as X, for
i = 1,..., T.
Let  m1, ..., mT  be  increasing sequence of real numbers.
Let   Yi= Xi + mi.
For any permutation s  of 1,2,..., T,  let P(s) =  P(Ys(1 ) <... < Ys(T))) .

Problem:  P(id) =  P(Y1 <... < YT)  >=   P(s)  for every permutation  s of 1,2,..., T.

Approximating  the distribution by a discrete distribution, we can assume, without loss of generality,
that  X takes n values  x1 < ... xn    with the same probability  1/n. Clearly, we can assume that  n  <= T,
x1=0, m1 = 0.
 

When  n =1, the inquality in  the problem becomes 1 > 0 for s =/ id, so
we can assume that n > 1.
 

When  T =2=n,   we have:

  1 = P(Y1 < Y2)  > 0 =   P(Y 2 <  Y1)   if   m2 >  x2,
 3/4 = P(Y1 < Y2)  > 0 =   P(Y 2 <  Y1)   if   m2 =  x2,
 3/4 = P(Y1 < X2)  > 1/4 =   P(Y 2 <  Y1)   if   m2 <  x2.

So the problem is solved for T = 2.
