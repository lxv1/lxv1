 1. Solve for x and y where a is a given number:

a2x + y= a2,

ax +ay = 1.

 

Solution. A row addition operation gives

a2x + y= a2,

(a - a3 )x = 1 - a3.

If a  ≠ 0 , ±1, then  x = (1 - a3 )/(a - a3 )=  (1+ a +a2 )/(a + a2 ) and

y = (1-x)a2 =  -a/(1+a).

If  a = 0 or -1, then  there are no solutions.

If a = 1, then y = 1 - x (x arbitrary).
