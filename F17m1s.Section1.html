4  4  4 


is  (A) 0, (B) 2, (C)  (3),  (D) 4.

14. Is [1,1,1,1]/4 optimal for the matrix game


1 0  2 3

0 2  2 2

2 1  3 0


(A) yes, (B) no,


Solution. We compute the payoff for S:


C1  C2   C3  C4    S

1    0       2     3    6/4

0    2       2     2    6/4

2    1       3      0   6/4


Name the players He and She.

If she uses  strategy  (C1+C2)/2 , she pays him 3/2 = 6/4 in the worst case. It is the same as S.

But  (C1+C2)/2 can be improved by a small step to C2. So S is not optimal.

B
