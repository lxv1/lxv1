 1. Solve for x, y   where a is a given number:

a2x - y= a2,

ax +ay = 1.


Solution. A row addition operation gives

a2x - y= a2,

(a + a3 )x = 1 +a3.

If a  ≠ 0 ,then  x = (1 +a3 )/(a +a3 )   and

y = (x - 1)a2 =  a(1 - a)/(1+a2).

If  a = 0 , then  there are no solutions.

max= 10 at  x = 1 y = ±3  (two optimal solutions).
